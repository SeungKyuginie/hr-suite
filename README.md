# HR 관리 시스템 (근태·휴가·급여) — 데모

한국 HR 실무 표준(근로기준법, 4대보험, 간이세액표)을 반영한 데모용 웹 인사관리.
**메모리 저장소** 기반이라 Vercel 한 곳에 그대로 올릴 수 있다.

## 빠른 시작 

```bash
# 로컬 개발
npm run setup
npm run dev          # http://localhost:5173

# Vercel 배포 (한 줄)
npx vercel --prod
```

자세한 배포 절차는 [DEPLOY.md](./DEPLOY.md) 참조.

## HR 전문가 방식 반영 포인트

### 근태 (Attendance)
- 1일 8시간 / 주 40시간 표준 근로 자동 분리
- 연장근로: 통상임금 ×1.5 가산
- 야간근로(22:00–06:00): ×0.5 추가 가산
- 휴일근로: 8시간 이내 ×1.5, 초과 ×2.0
- 휴게시간 자동 차감 (4h→30분, 8h→1h)

### 휴가 (Leave) — 근로기준법 제60조
- 입사 1년 미만: 매월 개근 시 1일 발생 (최대 11일)
- 입사 1년 이상: 15일 부여
- 3년차부터 매 2년마다 가산휴가 1일 (상한 25일)
- 결재 워크플로우: 신청 → 승인/반려 → 잔여 자동 차감

### 급여 (Payroll)
- 4대보험 자동 공제 (요율은 설정에서 수정 가능)
  - 국민연금 4.5%, 건강보험 3.545%, 장기요양 12.95%(건보료의), 고용보험 0.9%
- 근로소득세 누진세율 + 근로소득공제·세액공제 근사
- 지방소득세 = 소득세 × 10%
- 식대 월 20만원 비과세 자동 적용
- 급여명세서 출력 (브라우저 인쇄)

> ⚠️ 본 시스템은 데모용 참고 구현이며, 실 신고·공제는 세무사·노무사 검토를 권장한다.

## 시드 데이터

콜드 스타트 시 자동으로 다음 데이터가 로드된다:
- 직원 6명 (관리자 1명 포함, 다양한 근속연수)
- 각자 최근 14영업일 근태 (정시·연장·야간 케이스 혼합)
- 휴가 신청 4건 (대기·승인·병가 등 다양한 상태)

## 데모용 한계 (실 운영 전 보완사항)

| 항목 | 데모 상태 | 실운영 필요 작업 |
|---|---|---|
| 인증 | 없음 (셀렉터 전환) | JWT/세션 + bcrypt 비밀번호 |
| 저장소 | 메모리 (리셋됨) | PostgreSQL/Turso/Vercel KV |
| HTTPS | Vercel 자동 적용 | (그대로) |
| 개인정보 | 가상 데이터만 | AES 암호화 + 처리방침 |
| 백업 | 없음 | 일일 자동 백업 |
